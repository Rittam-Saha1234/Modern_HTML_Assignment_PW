1. Semantic Elements

Semantic elements are HTML elements that have a specific meaning or purpose. They are used to describe the structure of a document and the content of its elements.

Some examples of semantic elements are:

  • <header>
  • <fooret>
  • <section>
  • <article>
  • <aside>
  • <nav>

2. Multimedia Support

HTML5 allows you to embed video and audio files directly in your web pages. This makes it easy to create rich, interactive web pages.

Examples:

video:-

<video controls>
        <Source src="example.mp4" type="video/mp4">
        <Source src="example.webm" type="video/webm">
        
  • </video>
    •

    Audio:-

    <audio controls>
        <Source src="example.mp3" type="audio/mp3">
        <Source src="example.ogg" type="audio/ogg">
        
  • </audio>
    •

    3. Canvas API

    HTML5 includes canvas elements that allow you to draw on the page. This can be used to create games, animations, and other interactive content.

    Examples:-

    
        <canvas id="canvas" width="200" height="200"></canvas>

        <script>
            var canvas = document.getElementById("myCanvas");
            var ctx = canvas.getContext("2d");
            ctx.fillStyle = "black";
            ctx.fillRect(0, 0, canvas.width, canvas.height);
            ctx.fillStyle = "white";
            ctx.fillRect(100, 100, 100, 100);
        </script>
